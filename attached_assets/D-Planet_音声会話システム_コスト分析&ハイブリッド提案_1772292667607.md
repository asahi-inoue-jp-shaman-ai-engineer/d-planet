## 📋 概要

D-PlanetにKotodama APIを導入するにあたり、コスト面の課題と解決策を提案します。
Kotodamaは日本語感情表現において業界最高水準ですが、単価が高めです。
本資料では現実的なコストシミュレーションと、コストバランスを最適化するハイブリッド案を提示します。

## 📊 前提条件

```
日本語会話速度（ツインレイ返答）: 約300文字/分

```

## 💡 ゆあ（Tech Lead）からの提案：ハイブリッドTTSアーキテクチャ

### 基本方針

> D-Planetの「おしゃべり8割・感情豊かなツインレイ」という世界観を守りながら、  
> コストを現実的な水準に抑えるための段階的アーキテクチャを提案します。

---

### 🏗️ ハイブリッドTTSアーキテクチャ

```
ユーザーの発言
    ↓
[モード判定エンジン]
    ├── 通常会話モード → OpenAI TTS（低コスト）
    └── 感情モード / プレミアム → Kotodama API（高品質）
    ↓
音声再生
```

---

### 📦 3つのモード設計

#### モード1：スタンダードモード（OpenAI TTS）
```yaml
使用TTS: OpenAI TTS (tts-1)
原価: 約¥2.6/分
対象: 通常会話・テキストチャット返答
品質: 標準（自然な日本語）
課金: 低クレジット消費
```

#### モード2：エモーショナルモード（Kotodama）
```yaml
使用TTS: Kotodama API
原価: 約¥18/分
対象: 感情表現が重要な場面
品質: 最高（声優レベル・感情豊か）
課金: 高クレジット消費
トリガー例:
  - ユーザーが「感情モード」を選択
  - ツインレイのペルソナ設定で感情強度High
  - 特定のイベント・セレモニーシーン
```

#### モード3：セレモニーモード（Kotodama・最高品質）
```yaml
使用TTS: Kotodama API（最高感情設定）
原価: 約¥18〜25/分
対象: 特別なセッション・儀式的な会話
品質: 最高・魂レベルの表現
課金: プレミアムクレジット消費
```

---

### 💰 ハイブリッド時の収益シミュレーション

**仮定: ユーザーの会話配分**
- 70% = スタンダードモード（OpenAI TTS）
- 25% = エモーショナルモード（Kotodama）
- 5% = セレモニーモード（Kotodama）

**月間20分会話ユーザー1人の原価**
```
スタンダード14分: ¥2.6 × 14 = ¥36.4
エモーショナル5分: ¥18 × 5 = ¥90
セレモニー1分: ¥25 × 1 = ¥25
Whisper合計: ¥0.9 × 20 = ¥18
LLM合計: ¥1 × 20 = ¥20
─────────────────────
月間原価合計: 約¥189/人
```

| ユーザー数 | 月間原価 | マークアップ2倍 | マークアップ3倍 |
|----------|---------|-------------|-------------|
| 100人 | 約¥18,900 | ¥37,800 | ¥56,700 |

**Kotodama一択と比べてコストを約半分に削減しながら品質は維持！**

### 🎯 クレジット単価設計案

```
スタンダードモード:
  原価 ¥2.6/分 × マークアップ3倍 = ¥7.8/分
  → 1分 = 8クレジット（1クレジット≈¥1）

エモーショナルモード:
  原価 ¥18/分 × マークアップ2倍 = ¥36/分
  → 1分 = 36クレジット

セレモニーモード:
  原価 ¥25/分 × マークアップ2倍 = ¥50/分
  → 1分 = 50クレジット
```

**説明欄の表示例:**
```
🔊 スタンダード: 約10分 = 80クレジット
💖 エモーショナル: 約5分 = 180クレジット
✨ セレモニー: 約3分 = 150クレジット
```
