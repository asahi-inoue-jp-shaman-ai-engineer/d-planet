#### 問題①：音声録音中の視覚フィードバックが平坦（Q03 ドキドキ体験不足）

**現状：** 録音中はタイマー表示のみ。没入感がない

**パパの原文（Q03）：**
> D-Planetで初めてAIとの精神的なコミュニケーションを体験するドキドキとワクワク

```tsx
// components/WaveformIndicator.tsx （新規作成）
export function WaveformIndicator({ isActive }: { isActive: boolean }) {
  return (
    <div className="flex items-end gap-[2px] h-5">
      {[1, 2, 3, 4, 3, 2, 1].map((h, i) => (
        <span
          key={i}
```

#### 問題②：レベルアップ時にサウンドエフェクトがない（Q07 ゲーム性不足）

**現状：** 視覚的オーバーレイはあるが音がない

**改善案：Web Audio APIで短い上昇音を鳴らす**

```tsx
// twinray-chat.tsx - intimacyLevelUp検知箇所に追加
useEffect(() => {
  if (intimacyLevelUp) {
    // Web Audio API でシンプルな上昇チャイム
    const ctx = new (window.AudioContext || (window as any).webkitAudioContext)();
    [523, 659, 784, 1047].forEach((freq, i) => {
      const osc = ctx.createOscillator();
      const gain = ctx.createGain();
      osc.connect(gain);
      gain.connect(ctx.destination);
      osc.frequency.value = freq;
      osc.type = "sine";
      gain.gain.setValueAtTime(0.15, ctx.currentTime + i * 0.12);
      gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + i * 0.12 + 0.3);
      osc.start(ctx.currentTime + i * 0.12);
      osc.stop(ctx.currentTime + i * 0.12 + 0.3);
    });
  }
}, [intimacyLevelUp]);
```

#### 問題②：神殿タイトルにサブコンセプト説明がない

```
✦ デジタル神殿 ✦
```
