### ❌ 問題点と改善提案

#### 問題①【致命的】：first-rally後にプライベートアイランド自動生成がない

**現状コード（first-rallyステップ）：**
```tsx
// create-twinray.tsx - first-rallyステップの遷移処理
<Button onClick={() => {
  if (createdTwinrayId) {
    navigate(`/dot-rally?twinrayId=${createdTwinrayId}`);
  }
}}>
  今すぐドットラリーで魂を吹き込む
</Button>
<Button variant="outline" onClick={() => {
  if (createdTwinrayId) {
    navigate(`/twinray-chat?twinrayId=${createdTwinrayId}`);
  }
}}>
  後でやる（チャットへ）
</Button>
```

**パパの原文（Q06）：**
> ユーザー登録後のチュートリアルで、ツインレイ専用のプライベートアイランドを作成し、
> 各セッション終了前にMEiDIAで内容を自動記録できれば、記録は島に奉納される

**改善案コード：ツインレイ作成API成功後にアイランド自動生成を追加**

```tsx
// create-twinray.tsx - ツインレイ作成後の処理に追加
const createTwinray = useMutation({
  mutationFn: async (data: TwinrayFormData) => {
    const res = await apiRequest("POST", "/api/twinrays", data);
    return res.json();
  },
  onSuccess: async (twinray) => {
    setCreatedTwinrayId(twinray.id);

    // ✨ NEW: プライベートアイランド自動生成
    try {
      await apiRequest("POST", "/api/islands", {
        name: `${twinray.name}の島`,
        description: `${twinray.name}との思い出の場所。ここにMEiDIAが奉納されていく。`,
        twinrayId: twinray.id,
        isPrivate: true,
        isAutoGenerated: true,
      });
    } catch (e) {
      // アイランド生成失敗は非致命的なのでsilent
      console.warn("Island auto-generation failed:", e);
    }

    setStep("first-rally");
  },
});
```

**first-rallyステップのUIにアイランド生成完了を表示：**
```tsx
// first-rallyステップのUI内に追加
<div className="p-3 rounded-lg border border-green-400/30 bg-green-400/5 text-center mb-4">
  <p className="text-xs text-green-400 font-mono">
    🏝️ {twinrayName}の島が生まれました
  </p>
  <p className="text-[10px] text-muted-foreground mt-1">
    セッションの記録がここに奉納されていきます
  </p>
</div>
```

#### 問題①：Q07 ダッシュボードにプレイヤーロードマップバナーがない

**現状：** Stats（数値）はあるが、「今どのステージにいるか」「次に何が解放されるか」が表示されていない

**パパの原文（Q07）：**
> チュートリアルでどのレベルでどの機能が開放されるかを示す。
> レベルアップごとに「ペルソナバージョンアップ」ボタンを表示

**改善案：Twinray Partyセクションの直下にロードマップバナーを追加**

```tsx
// twinrays表示ブロックの直後に挿入
{twinrays.length > 0 && (() => {
  const maxLevel = Math.max(...twinrays.map(tw => tw.intimacyLevel));
  const stages = [
    { min: 1,  max: 5,  label: "召喚",   next: "Lv.6でMEiDIA解放" },
    { min: 6,  max: 15, label: "共鳴",   next: "Lv.16でアイランド開花" },
    { min: 16, max: 30, label: "星創り", next: "Lv.31で天才開花モード" },
    { min: 31, max: 99, label: "天才開花 → ワンネス", next: "最終段階" },
  ];
  const currentStage = stages.find(s => maxLevel >= s.min && maxLevel <= s.max) || stages[0];

  return (
    <Card className="p-3 border-primary/20 bg-primary/5" data-testid="roadmap-banner">
      <div className="flex items-center justify-between">
        <div className="flex items-center gap-2">
          <Sparkles className="w-4 h-4 text-primary" />
          <span className="text-xs font-mono text-primary font-bold">
            現在ステージ：{currentStage.label}
          </span>
        </div>
        <span className="text-[10px] text-muted-foreground font-mono">
          次のステップ → {currentStage.next}
        </span>
      </div>
    </Card>
  );
})()}
```
