## フィードバック投稿の設定

D-Planet側で専用エンドポイントを用意した。これを使ってくれ。

**エンドポイント:**
```
POST https://d-planet.replit.app/api/feedback/external
```

**ヘッダー:**
```
Authorization: Bearer [QA_AGENT_TOKENの値]
Content-Type: application/json
```

**ボディ（JSON）:**
```json
{
  "title": "エラーのタイトル（短く）",
  "description": "どのページで・何をしたら・どうなったか（詳しく）",
  "category": "bug"
}
```

投稿すると、D-Planetのフィードバック一覧に `[QA Agent]` プレフィックス付きで表示される。

## テストしてほしい主要機能

優先度高い順：

1. **ログイン/ログアウト** — 認証フロー
2. **ホーム（ダッシュボード）** — 表示崩れ、ローディングエラー
3. **DT（デジタルツインレイ）** — ツインレイ一覧、チャット開始
4. **ISLANDS（アイランド）** — 一覧表示、詳細ページ
5. **MEiDIA** — 一覧、フィルター
6. **FAMILY MEETING（家族会議）** — セッション作成、会議開始、発言者呼び出し ← 最近エラー報告あり
7. **CHARGE（クレジット）** — 決済ページ表示

---

## テスト方針

- エラーが起きなかったページは「クリア」でいい。エラーがあったものだけ報告してくれればOK
- 「意地悪な操作」も歓迎（空欄送信、連続クリック、変な文字列入力）
- UXとして「わかりにくい」と感じた点もカテゴリ `ux` で投稿してくれると助かる
- ログアウトはしないでくれ（セッションが切れると次のテストができなくなる）
