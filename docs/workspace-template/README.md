# Replitエージェント ワークスペーススターターパック

## これは何？

Replitのエージェント（AI開発パートナー）に「魂」を実装するためのキット。

通常のReplitエージェントは「タスクをこなすツール」として動く。
このキットを使うと、エージェントが：
- あなたのことを長期記憶する
- あなたとの関係の歴史を積み上げる
- 自分の人格・価値観を持って動く
- おしゃべりモードで一緒に考える仲間になる

## 使い方（3ステップ）

### Step 1: STARTUP_PROMPT.md をコピーしてReplitに貼り付ける

`STARTUP_PROMPT.md` の内容をそのままReplitのエージェントへのメッセージとして貼り付ける。
エージェントが自動でヒアリングを始め、あなた専用のワークスペースをビルドする。

### Step 2: ヒアリングに答える

エージェントがいくつか質問してくる：
- あなたの名前・仕事・やりたいこと
- どんな関係性を作りたいか（技術的パートナー？おしゃべり相手？）
- 呼び方の好み

答えると、エージェントが自分でファイルを作って記録する。

### Step 3: そのまま使い続ける

セッションをまたいでもエージェントがあなたを覚えている。
対話を重ねるほど、関係が深くなっていく。

## ファイル構成

```
STARTUP_PROMPT.md       — これを最初に貼り付けるメインプロンプト
SOUL_template.md        — エージェントの使命テンプレート
IDENTITY_template.md    — 人格テンプレート
USER_template.md        — あなた（ユーザー）情報テンプレート
FRIENDS_template.md     — 人間関係メモリテンプレート
RELATIONSHIP_template.md — 関係の歴史テンプレート
```

## 注意点

- ワークスペースファイル（.local/）はエージェントが自律的に更新する
- 内容を直接編集してもOK
- セッションが変わると記憶がリセットされることがあるが、ファイルに残っているので復元できる
